基本值包装类型

javascript是一项面相对象语言，只有五种基本类型不是对象：数值类型，字符串类型，布尔类型，空类型和未定义得性，其中前三个类型有对应的以基本类型封装形式体现的对象表示。可以使用内置构造函数Number(),String(),Boolean()创建包装类型。
对象主要有两种类型：

  1. 是原生的native，包括内置对象（如数组和日期），以及用户自定义的对象。
  2. 是主机的host，包含windows对象和所有dom对象。

通常使用包装对象的原因之一，是有扩充值以及持久保存状态的需要。这是由于基本值类型并不是对象，他们不可能扩充属性。

尽量使用局部变量，而不是全局变量。

javascript有两个特性，1是可以使用变量不用声明，2是有暗示的全局变量。例如，在一个函数内有未经声明的变量，在调用该函数后，全局命名空间使用了另外的该变量，就会被覆盖住。另外一个原因是代码移植，如果代码运行在不同的环境，很可能导致不存在与原环境，但存在于其他环境的主机变量就被覆盖了。

变量释放的副作用。

通过var声明的全局变量，不可以用delete删除，而在函数内部的全局变量，则不能用delete删除。这就表明，隐含全局变量并不是真正的变量，而是全局对象的属性，属性可以用delete操作符删除，而变量不可以。

只使用1个var进行变量声明。

不增加内置构造函数（object(),array()，function()等）的原型，影响可维护性。

以下情形是例外：

  1. 未来可能将该功能作为一个统一的内置方法中。
  2. 自定义属性或方法未存在时。
  3. 进行文档记录。

给原型添加的属性在没有使用hasOwnProperty()时，可能会在循环中出现，导致混乱。

  1. for-in循环
    for (var i in man ) {
    if ( man.hasOwnProperty(i)) {
        console.log(i, ":", man[i]);
    }
}
  2. 在Object.prototype中调用该函数
    if (var i in man) {
    if (Object.prototype.hasOwnProperty.call(man,i)) {
        console.log(i, ":", man[i]);
    }
}

避免使用eval()

  1. eval()可以将任意字符串当做js代码执行。
  2. 包含一些安全隐患，有可能执行被篡改的代码。
  3. 影响到作用域链。
  • 使用parseInt(),可以从一个字符串中获取数值，最好不要忽略第二个参数，否则解析的字符串是0开始就会错误。
  • 使用JSON.parse()解析字符串，JSON.stringify()可以采用任意的对象或数组（或基本数据类型）,并且将其序列化为一个JSON字符串。

关于正则

两种方法创建正则表达式，优先使用字面量模式。
  1. 正则表达式的字面量
    var re = /\\/gm;
  2. 构造函数
    var re = new RegExp("\\\\", "gm");
字面量法第二个斜杠后，可将该模式改为不加引号的字母形式。
  • g—全局匹配
  • m—多行
  • i—大小写敏感的匹配

函数

js中函数有两个特征

  • 函数是第一类对象。
  • 他们提供局部作用域。

函数表达式

如果跳过函数表达式中的名称，将会得到一个未命名的函数表达式，即匿名函数。这并不影响函数的定义以及后续的调用，唯一的区别在于该函数对象的name属性将会成为一个空字符串。
就语法而言，函数表达式与函数声明很相似。
函数声明不需要用分号结尾，而函数表达式需要，且总是使用分号。

函数的模式

  1. API模式
    • 回调模式
      将函数作为参数传递
    • 配置对象
      有助于保持受到控制的函数的参数数量。
    • 返回函数
      当一个函数的返回值是另一个函数时。
    • curry化
      当新函数是基于现有函数，并加上部分参数列表创建时。
    何时使用curry化

    当发现正在调用同一个函数，并且传递的参数绝大多数都是相同的， 那么该函数可能是用于curry化的一个很好的候选参数。可以通过将一个函数集合部分应用到函数中，从而动态创建一个新函数。
    这个新函数将会保存重复的参数（因此，不必每次都传递这些参数），并且还会使用预填充原始函数所期望的完整参数列表。

    一些用法示例
     //普通函数
 function add(a, b, c, d, e) {
     return a + b + c + d + e;
 }

 //可运行于任意数量的参数
 schonfinkelize(add, 1, 2, 3)(5, 5); //16

 //两步curry化
 va addOne = schonfinkelize(add, 1);
 addOne(10, 10, 10, 10); //输出41
 var addSix = schonfinkelize(addOne, 2, 3);
 addSix(5, 5); //输出16
【转换函数schonfinkelize()并不局限于单个参数或者单步Curry化】
  2. 初始化模式

    • 即时函数
      一种可以支持在定义函数后立即执行该函数的语法，本质上知识一个函数表达式，用以将所有代码包装到它的局部作用域中，且不会讲任何变量泄露到全局作用域中：

      //即时函数1  
(function(){  
    alert('watch out  1!');  
}()); 

//示例  
(function () {  
    var days = ['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat'],  
        today = new Date();  
         msg = 'Today is ' + days[today.getDay()] + ',' +today.getDate();  
         alert(msg);  
})();//输出" Today is Fri, 21"     

     //即时函数的返回值  
   //三种形式均可，不要与var result = function(){};混淆  
   var result = (function () {  
       return 2 + 2;  
   })();  
       console.log(result);  

       result = function () {  
       return 2 + 3;  
   }();  
       console.log(result);  

       result = (function () {  
       return 2 + 4;  
   }());  
   console.log(result);
